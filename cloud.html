Clouds, services, servers

What does it do?

Cloud is everywhere in our life, which brings us a lot of benefits. What is cloud? Cloud is working via the internet. For example, cloud is able to host and deliver services over the internet. Unlike traditional hosting, cloud hosting makes applications and websites accessible using cloud resources. For example, use a network of connected virtual and physical cloud servers hosting the website or application, the adventager is flexibility and scalability. Cloud services have three main delivery models, which are Software as a Service (SaaS), Infrastructure as a Service (IaaS) and Platform as a Service (PaaS). SaaS is a way of delivering applications over the internet as a service, such as email, Netflix, Microsoft Office, those are common applications in our daily life. Therefore, users no longer need to install and maintain software, they can just simply access it via the internet and be free to manage their software and hardware. IaaS is a system of cloud computing that delivers virtualized computing resources over the internet, which means businesses are able to use IaaS to provide the infrastructure to support web apps, such as storage, web and application services, networking resources. Using IaaS technology is able to increase security, scalability and flexibility. PaaS is a cloud environment in which customers are able to complete development and deployment. In other words, PaaS is a platform or tool for building applications or websites via the internet. For instant, operating system, programming language execution environment. The difference between PaaS and SaaS is that PaaS is the cloud-based platform for building software. SaaS products are fully developed by other companies for users to use via the internet.
