Skills and Jobs

Yonathan Kogan

In order for the project to develop successfully, the team would need a senior software engineer, a data scientist, a marketing coordinator, and a UX designer. The senior software engineer would be responsible for the programming and coding part of the whole project and ensuring that the website works as intended, the data scientist would be analysing the data and if there is a problem regarding the whole project, they should also be making the correct decision to solve the problem. The marketing coordinator is needed so that the project is able to gain more users, and lastly a UX designer would be responsible for the interface of the website and if users do not like the way the website looks and functions then the UX designer should come up with a solution so that the website can satisfy users.

Harshnoor Singh

To help us reach our desired final product, we would need to hire a data scientist and a UX designer. A data scientist is required in-case we encounter a hard decision. A data scientist will be able to use data and interpret the said data to help us understand the problem that requires us to make a decision. As a result of this, the decision we make will be more informed as we are intending to take a more data driven approach in our organisation. Furthermore, as one of our ultimate aims is to make a beginner friendly application, a UX designer will be necessary to help understand our user base in more details and will help us determine how we can optimise our user interface to better cater to our user base.
